# Estruturas de Dados em C

Implementações práticas de estruturas de dados desenvolvidas em C, com foco em alocação dinâmica e manipulação manual de memória.

## Conteúdo

- Pilhas
- Filas
- Listas encadeadas

## Conceitos utilizados

- `struct`
- ponteiros
- `malloc` / `free`

## Objetivo

Exercitar a implementação manual das estruturas e compreender seu funcionamento interno.
